Правила информационной безопасности

Инструкции для сотрудников Арены Кузнецких металлургов

1. Общие положения

Настоящие правила обязательны для соблюдения всеми сотрудниками арены, имеющими доступ к корпоративной сети и автоматизированным рабочим местам (АРМ).

  • Не передавайте свои учетные данные третьим лицам.
  • Обо всех подозрительных инцидентах немедленно сообщайте в ИТ-отдел.

2. Парольная безопасность

  • Создавайте сложные пароли
  • Минимум 8 символов (рекомендуется 12+)
  • Используйте буквы в верхнем и нижнем регистре, цифры, специальные символы
  • Не используйте личную информацию (имена, даты рождения)
  • Не записывайте пароли на бумаге и не храните в открытом виде на компьютере
  • Не передавайте пароли коллегам по телефону или в мессенджерах
  • Используйте корпоративный менеджер паролей при его наличии
  • Регулярная смена паролей
  • Меняйте пароли каждые 3 месяца
  • Не используйте повторно старые пароли
  • Немедленно меняйте пароль при подозрении на компрометацию

3. Безопасность электронной почты

  • Проверяйте отправителей
  • Внимательно проверяйте адрес электронной почты отправителя
  • Обращайте внимание на орфографические ошибки в доменных именах
  • Не доверяйте письмам с подозрительных адресов
  • Осторожность с вложениями и ссылками
  • Не открывайте вложения из писем от неизвестных отправителей
  • Наводите курсор на ссылки для проверки настоящего URL перед переходом
  • Скачивайте файлы только из доверенных источников

    • Защита от фишинга

  • Не отвечайте на письма с запросом паролей или личной информации
  • Сообщайте в ИБ-отдел о любых подозрительных письмах
  • Помните: официальные службы никогда не запрашивают пароли по email

4. Безопасность рабочего места

    Физическая безопасность

  • Всегда блокируйте компьютер при отходе от рабочего места (Win+L)
  • Не оставляйте конфиденциальные документы на столе без присмотра
  • Используйте замки для шкафов и ящиков с важными документами

    • Управление доступом

  • Не предоставляйте доступ к своему компьютеру посторонним лицам
  • Своевременно сообщайте о увольнении сотрудников для отзыва доступов
  • Не используйте учетные записи коллег для выполнения задач

    • Работа вне офиса

  • Используйте VPN при подключении к корпоративной сети извне
  • Не работайте с конфиденциальными данными в публичных местах
  • Обеспечивайте приватность экрана в общественных местах

5. Работа с данными и файлами

    Классификация данных

  • Знайте уровень конфиденциальности обрабатываемых данных
  • Храните файлы в соответствии с их классификацией
  • Не передавайте конфиденциальные данные без необходимости

    • Безопасное хранение

  • Используйте только утвержденные корпоративные хранилища
  • Не храните рабочие файлы на личных облачных сервисах
  • Регулярно создавайте резервные копии важных данных

    • Передача данных

  • Используйте шифрование при передаче конфиденциальной информации
  • Не отправляйте важные данные через личные мессенджеры
  • Убедитесь в правильности адресата перед отправкой файлов

6. Интернет-безопасность

    Браузинг и загрузки

  • Посещайте только доверенные веб-сайты
  • Не скачивайте программное обеспечение из непроверенных источников
  • Используйте только одобренные браузеры и расширения

    • Социальные сети

  • Не раскрывайте рабочую информацию в личных профилях
  • Остерегайтесь запросов на добавление в друзья от незнакомцев
  • Соблюдайте корпоративную политику использования соцсетей

    • Устройства и носители

  • Не подключайте личные USB-устройства к рабочим компьютерам
  • Используйте только корпоративные внешние носители
  • Проверяйте устройства на вирусы перед использованием

7. Программное обеспечение и обновления

    Установка ПО

  • Устанавливайте программы только с разрешения ИБ-отдела
  • Не используйте нелицензионное программное обеспечение
  • Сообщайте о необходимости установки нового ПО через официальный запрос

8. Противодействие социальной инженерии

    Телефонная безопасность

  • Проверяйте личность звонящих перед предоставлением информации
  • Не сообщайте внутреннюю информацию по телефону без авторизации
  • Перенаправляйте подозрительные звонки в ИБ-отдел

    • Посетители и гости

  • Не пропускайте неидентифицированных посетителей в офис
  • Сопровождайте гостей в пределах офиса
  • Сообщайте службе безопасности о подозрительных лицах

    • Утилизация документов

  • Уничтожайте конфиденциальные документы с помощью шредера
  • Не выбрасывайте рабочие бумаги в общие мусорные корзины
  • Следуйте процедурам безопасной утилизации носителей информации

9. Действия при инцидентах безопасности

    Обнаружение угроз

  • Немедленно сообщайте о подозрительной активности в ИБ-отдел
  • Сохраняйте доказательства (скриншоты, письма, логи)
  • Не пытайтесь самостоятельно устранить сложные инциденты

    • План действий

  • Отключайте компьютер от сети при признаках заражения
  • Следуйте инструкциям ИБ-отдела при расследовании инцидентов
  • Участвуйте в пост-инцидентном анализе для улучшения безопасности

    • Чрезвычайные ситуации

  • Знайте процедуры эвакуации и спасения данных
  • Имейте доступ к критически важным контактам
  • Соблюдайте спокойствие и следуйте указаниям ответственных лиц
